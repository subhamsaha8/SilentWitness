# 🛡️ Silent Witness - Secret Message Encoder/Decoder

Silent Witness is a security-focused Android application designed to protect private communication. It allows users to encrypt plain text into a secure format and decrypt it back, ensuring that sensitive information remains confidential.

## 🔑 Key Features

*   **Custom Encryption Logic:** Implements a unique algorithm to transform messages into secure codes.
*   **Real-time Decoding:** Quick and seamless decryption of messages back to their original form.
*   **Intuitive UI:** A simple, clean interface for easy message handling, designed with a focus on user experience.
*   **Privacy Centric:** No data is stored externally; all processing happens locally on the device.

## 🛠️ Tech Stack

*   **Language:** Java
*   **Platform:** Android Studio
*   **Design Concepts:** Logical Algorithm Development, UI/UX Design (Figma), Mobile Security
